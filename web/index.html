Tehnologii web

În decursul laboratoarelor de la Tehnologii Web o parte dintre tematicilile ce vor fi abordate sunt:

  1. Concepte importante și arhitectura WWW
  2. Protocolul HTTP + cookie-uri și sesiuni
  3. Limbajul de marcare HTML + foi de stiluri CSS
  4. Inginerie Web: proiectare și arhitecturi Web, MVC et al.
  5. Programare Web: servere de aplicații – exemplu: Python / PHP
  6. Modelarea datelor via XML – XPath, DTD
  7. Procesări DOM, SAX, SimpleXML pentru HTML și XML
  8. De la SOA la servicii Web: SOAP și REST
  9. Interacțiune Web: Ajax + aplicații hibride (mash-ups)
  10. Securitatea aplicațiilor Web

Notă:
Resursele disponibile complementează şi nu substituie prezenţa la cursuri şi laboratoare.
În funcție de gradul de interes, tematicile și calendarul pot varia.
